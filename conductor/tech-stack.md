# Tech Stack

## Language & Core
*   **Language:** TypeScript (v5.7.3)
*   **Runtime:** Node.js
*   **Framework:** NestJS (v11.0.1)

## Data & Persistence
*   **Database:** PostgreSQL
*   **ORM:** TypeORM (v0.3.28)
*   **Migrations:** TypeORM CLI
*   **Naming Strategy:** Custom naming strategy (snake_case)

## Infrastructure & Services
*   **Caching:** @nestjs/cache-manager with cache-manager (v7.2.7)
*   **Mailing:** @nestjs-modules/mailer with Resend integration
*   **Validation:** class-validator and Zod (v4.3.5)
*   **Authentication:** Passport.js with JWT and Argon2 hashing

## Testing & Quality
*   **Testing Framework:** Jest (v30.0.0)
*   **E2E Testing:** Supertest
*   **Linting:** ESLint with Prettier

## Key Libraries
*   **Payments:** Stripe (v20.1.2)
*   **File Processing:** XLSX (v0.18.5)
*   **Date Handling:** date-fns (v4.1.0)
*   **Utility:** uuid, reflect-metadata, rxjs
